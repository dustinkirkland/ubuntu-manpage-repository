Welcome to the Ubuntu Manpage Repository

This site contains hundreds of thousands of dynamically generated manuals, extracted from every package of every supported version of Ubuntu, and updated on a daily basis. Traditionally, manpages are browsed on the command line. This project renders all such manuals included in Ubuntu in an HTML, web-browsable format.

Navigating the Repository

  • Browse the repository as a hierarchical file structure
  • Use the search form at the top of every page
  • Install the search engine plugin for your browser
  • Download the 'dman' shell script to run on a command line
